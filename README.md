# Frontend Mentor - 3 Column Preview Card Component

Esse projeto foi desenvolvido por mim como solução do desafio
[3-column preview card component](https://www.frontendmentor.io/challenges/3column-preview-card-component-pH92eAR2-)
do Frontend Mentor.

O objetivo foi construir um componente de cards responsivo, seguindo o layout proposto no desafio,
com boa semantica HTML, organizacao de CSS e estados de interacao.

## Visao geral

### O desafio

Os usuarios devem conseguir:

- Visualizar o layout ideal de acordo com o tamanho da tela
- Ver o estado de hover nos botoes

## Meu processo

### Tecnologias utilizadas

- HTML5 semantico
- CSS3
- CSS custom properties (variaveis)
- Flexbox
- Grid Layout
- Mobile-first workflow

### O que eu pratiquei nesse projeto

- Estruturacao de componentes em cards reutilizaveis
- Organizacao de tokens de design com :root
- Responsividade para mobile e desktop sem quebrar o layout
- Ajuste de hover/focus para melhor experiencia de uso

### Aprendizados

Nesse desafio, eu reforcei principalmente a importancia de:

- Definir variaveis de cor e espacamento no inicio do CSS
- Trabalhar primeiro no layout mobile e depois escalar para desktop
- Separar bem a estrutura (HTML) da apresentacao (CSS)
